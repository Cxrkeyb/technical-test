## Technologies
- NextJS
- TailwindCSS
- JavaScript
- HTML
- CSS

## You have to need
- Node.js
- NPM
- Git
- Code editor
- Browser

## Installation
1. Clone the repository
```bash
git clone
```
2. Install dependencies
```bash
npm install
```
3. Start the development server
```bash
npm run dev
```
4. Open the browser and navigate to `http://localhost:3000`
5. Enjoy!
